# Chat Console Examination

## how it works

chatroom set by default to general

When starting the program it will ask for username. After typing username it will connect for you to the server. Username can include numbers. When connected it will display chat history of 20 last messages. 
When you type something it will send indicator to the other users that youre typing. When you send message it will display in chat history. you are able to chat with anyone in the same chatroom. chatroom is general

## Commands

- /start: connects to the server. When chat app starts and typing username it will automatically connect for you. /start can be used to reconnect after disconnecting.
- /help: shows all commands.
- /leave: disconnects from the server and leaves the chat.
- /exit: closes the program.
